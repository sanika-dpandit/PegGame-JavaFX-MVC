# 🎮 Peg Game – JavaFX MVC Project

A JavaFX-based Peg game implemented using MVC architecture.

## Features
- JavaFX GUI with interactive peg board
- MVC architecture (Model–View–Controller)
- Load and Save game states from file
- Pause / Resume gameplay
- Move validation logic
- Win and Stalemate detection
- Clean OOP design

---

## Technologies Used
- Java 17+
- JavaFX
- Object-Oriented Programming
- File Handling (BufferedReader / BufferedWriter)

---

## Architecture

- **Model:** Game logic, board state, move validation
- **View:** JavaFX UI (buttons, grid, labels)
- **Controller:** Connects UI actions to model logic
